Frontend Development

Front-end web development is the development of the graphical user interface of a website through the use of HTML , CSS, and JavaScript so users can view and interact with that website.


  • HTML:Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser.

  • CSS:Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language


    1. pure CSS

    2. bootstrap(framework)

    3. tailwind(framework)

    4. sass(preprocessor)

  • JavaScript:JavaScript is a programming language used for creating dynamic content on websites




    1. vanilla JS

    2. jquery

    3. react

    4. angular

    5. typescript
